# Resultados-Implementación-3DGD-en-objetos-arqueológicos
Este repositorio contiene los modelos 3D entrenados, conjuntos de datos y scipts de procesamiento y resultados

Guía de Uso del Visor Web de Modelos 3D

Esta guía te guiará a través del proceso para cargar y navegar modelos 3D utilizando el visor web antimatter. Sigue los pasos cuidadosamente para obtener la mejor experiencia de visualización.

Paso 1: Descarga de Archivos

En la carpeta Proyecto VRI ID 6292 entra en el directorio Modelos entrenados contiene los archivos de cada modelo para tres volantes de huso. Aquí los podrás ver Modelos entrenados
Descarga los siguientes archivos para cada volante de uso:
point_cloud.ply: Este archivo contiene la nube de puntos del modelo 3D.
cameras.json: Este archivo contiene las configuraciones predefinidas de las cámaras para visualizar el modelo desde diferentes ángulos.

Paso 2: Acceso al Visor Web

Abre el visor web en tu navegador:
Ingresa a la siguiente URL: https://antimatter15.com/splat/

Paso 3: Cargar los Archivos en el Visor

Arrastra los archivos al visor:
Abre la carpeta donde descargaste los archivos.
Arrastra y suelta el archivo point_cloud.ply dentro de la ventana del visor en tu navegador. Esto cargará el modelo 3D.
Luego, arrastra y suelta el archivo cameras.json dentro del visor para cargar las vistas de cámara predefinidas.

Paso 4: Navegar por el Modelo
Controles de navegación:
 Una vez que los archivos estén cargados, puedes empezar a explorar el modelo utilizando los siguientes controles:

Teclado:

Flechas (↑, ↓, ←, →):
Teclas de flecha izquierda/derecha para desplazarse lateralmente
Teclas de flecha arriba/abajo para avanzar/retroceder
Espacio para saltar

Ángulo de cámara (W/S)
A/D para girar la cámara a la izquierda/derecha
W/S para inclinar la cámara hacia arriba/abajo
Q/E para girar la cámara en sentido horario/antihorario
I/K y J/L para orbitar

0-9: Cambia entre las vistas predefinidas de cámara que están cargadas desde el archivo cameras.json.

Rueda del ratón o trackpad:

Zoom: Usa la rueda del ratón o desliza con dos dedos en el trackpad para acercar o alejar la vista del modelo.

Ratón:

Click y arrastrar: Mantén presionado el botón izquierdo del ratón y arrastra para rotar el modelo en cualquier dirección.
Ctrl + Click y arrastrar: Mantén presionado la tecla Ctrl mientras mantienes presionado el botón izquierdo del ratón para hacer zoom (acercar o alejar el modelo)
